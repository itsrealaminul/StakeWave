# 🌊 StakeWave — Telegram Mini App

**Predict. Stake. Earn Real Value.**

A Telegram Mini App for prediction markets, staking, and earning — targeting US/EU markets with zero investment.

## 🚀 Features

- 🔮 **Prediction Markets** — Predict crypto prices, sports, events
- 💰 **Staking** — Earn 15% APY on your points
- 📋 **Tasks** — Complete offers, watch ads, earn rewards
- 👥 **Referrals** — 12% commission forever
- 🎡 **Daily Spin** — Free daily rewards
- 🏆 **Leaderboard** — Compete with top earners
- 💸 **Withdraw** — PayPal, USDT, TON

## 🛠️ Tech Stack

- **Backend:** Python (FastAPI + aiogram 3.x)
- **Frontend:** HTML/CSS/JS (Telegram Mini App SDK)
- **Database:** SQLite (PostgreSQL ready)
- **Hosting:** Vercel (frontend) + Railway (backend)
- **Cost:** $0/month (free tier)

## 📦 Quick Start

```bash
# Backend
cd backend
pip install -r requirements.txt
cp .env.example .env
# Add your bot token to .env
python main.py

# Frontend
cd frontend
# Open index.html or serve with any static server
```
